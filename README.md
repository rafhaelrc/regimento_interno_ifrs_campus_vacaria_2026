# Regimento Complementar — IFRS Campus Vacaria
## Análise Comparativa e Proposta de Atualização

Documento interativo comparando o Regimento Complementar do Campus Vacaria com os regimentos de outros campi do IFRS, com sugestões de texto atualizadas.

---

## 📂 Estrutura do projeto

```
regimento-vacaria/
├── index.html              ← Documento principal (abrir no navegador)
├── README.md               ← Este arquivo
└── arquivos-fonte/
    ├── text_Vacaria.txt    ← Texto extraído (OCR) do regimento do Vacaria
    ├── text_Bento.txt      ← Regimento Bento Gonçalves (2024)
    ├── text_Erechim.txt    ← Regimento Erechim (2024)
    ├── text_Feliz.txt      ← Regimento Feliz (2025)
    ├── text_Ibiruba.txt    ← Regimento Ibirubá (2025)
    ├── text_Canoas.txt     ← Regimento Canoas (2018)
    ├── vacaria_sections.json        ← Seções do Vacaria estruturadas por setor
    └── other_campus_sections.json  ← Adições compiladas dos outros campi
```
